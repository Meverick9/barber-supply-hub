# Barber Supply Hub (MVP)

Next.js App Router project scaffold with:
- Picks pages generated from JSON
- Product cards, comparison table, sticky CTA
- GA4 page_view + outbound_click + scroll depth tracking
- UGC placeholders: barber profiles, tool stacks, votes, reviews
- Reputation + ranking + personalization + wear model (libs)

## Run
1) `npm i`
2) Create `.env.local`:
   - `NEXT_PUBLIC_GA_ID=G-XXXXXXXXXX`
3) `npm run dev`
